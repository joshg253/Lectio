Instance settings

Maintenance & cleanup

Runs nightly at the set hour (server local time): prunes feed fetch history, applies per-folder retention, sweeps tombstones, and evicts the image cache.{% if maintenance_last %} Last ran: {{ maintenance_last }}.{% endif %}

Image cache

The /api/img proxy caches fetched images. Entries not viewed within the retention window are evicted nightly; larger images are downscaled to the max dimension before storing.

Security

Repeated failed logins lock the account out for the window.

New users

Seeds the auto-refresh setting for newly created users.

Shared keys

Instance-wide credentials. The email key sends on behalf of the whole instance; the OAuth apps are a shared fallback for users who haven't set their own (a user's own credentials always take priority).

Email (Resend)

One sending account for the instance (only a verified domain can send).

Shared OAuth apps (optional)

YouTube: Register at Google Cloud Console. Add {{ public_url }}/youtube/oauth/callback as an authorized redirect URI.

Pinterest: Register at Pinterest Developers. Add {{ public_url }}/integrations/pinterest/oauth/callback as a redirect URI.

Reddit: Register a web app at reddit.com/prefs/apps. Add {{ public_url }}/integrations/reddit/oauth/callback as the redirect URI.
